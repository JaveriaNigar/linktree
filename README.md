# Javeria Nigar Linktree

This repository contains the **HTML and CSS** for my personal Linktree page, where you can access all my social media profiles and portfolio links in one place.

## Features
- Clean and modern design
- Responsive for mobile and desktop
- Easy to update links
- Hover effects for interactive buttons
- Professional header and footer

## Files
- `index.html` → Main page structure
- `style.css` → Styling for layout, buttons, header, and footer

## Usage
1. Clone or download this repository
2. Open `index.html` in your browser
3. Modify links in HTML as needed
